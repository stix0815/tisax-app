## 🎯 Assessment Logic (CORRECTED)

**Prerequisites - ALL must be filled:**
- k1: Infosec (Vertraulichkeit ≠ Off AND Integrität ≠ Off AND Verfügbarkeit ≠ Off)
- k2: Datenschutz (Personenbezogene ≠ Off AND Besondere ≠ Off)
- k3: Prototypenschutz (Bauteile ≠ Off AND Fahrzeuge ≠ Off AND Erprobung ≠ Off AND Events ≠ Off)

**If NOT (k1 AND k2 AND k3):** No result shown (form incomplete)

**If (k1 AND k2 AND k3):**
1. **AL3** (Sehr hoher Schutzbedarf): ANY Infosec = "Sehr hoher"
   - → TISAX erforderlich: JA + detailed requirements text

2. **AL2** (Hoher Schutzbedarf): ANY Infosec = "Hoher"
   - → TISAX erforderlich: JA + detailed requirements text

3. **AL1** (Normal/Kein Schutzbedarf): All other Infosec combinations
   - IF any Datenschutz OR Prototyp = "Ja"
     - → TISAX erforderlich: JA + text
   - ELSE (all = "Nein")
     - → "Kein TISAX-Zertifikat notwendig"
