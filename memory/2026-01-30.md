- Input: Ticker symbol (e.g., AAPL, TSLA)
- Output: Bullish & bearish scenarios with buy/sell recommendation

**Planned Components:**

- Project structure setup
- Data sources: yfinance, Alpha Vantage, SEC EDGAR API
- Modular design

### Phase 2: Technical Indicators
- **Trend:** MACD, SMA (50/200), EMA, golden/death cross
- **Momentum:** RSI, Stochastic Oscillator
- **Volatility:** Bollinger Bands, ATR, band squeeze
- **Volume:** Volume profiles, OBV

### Phase 3: Probabilistic Models
- Monte Carlo price simulations
- Bayesian probability updates
- Mean reversion detection

- SEC filing scraper (13D/13F)

- Order book analysis (bid-ask spreads, liquidity)
- VIX/Tick Index correlation

**Key Requirements:**
