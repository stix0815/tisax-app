# Finanz-Übersicht Google Sheet - Manuelle Einrichtung

Da `gog` CLI keine direkte Sheets-API-Unterstützung hat, hier die schnellste Methode:

## Schritt 1: Sheet erstellen (2 Minuten)

1. Geh zu: https://sheets.google.com
2. **Neues Sheet** → Titel: "Finanz-Übersicht"
3. Erstelle **4 Tabs**:
   - Dashboard
   - 2026
   - Kategorien  
   - Statistiken

## Schritt 2: Verschieben nach Atlas Workspace

1. Im Sheet: **Datei → Verschieben**
2. Wähle: **Atlas Workspace/Dokumente**

## Schritt 3: Sheet-ID kopieren

1. Aus der URL: `https://docs.google.com/spreadsheets/d/SHEET_ID/edit`
2. Kopiere die **SHEET_ID**
3. Schick sie mir im Chat

## Dann mache ICH:

✅ Alle Headers einfügen  
✅ Kategorien mit Budgets  
✅ Formeln für Statistiken  
✅ Data Validation (Dropdowns)  
✅ Conditional Formatting  
✅ Charts im Dashboard
